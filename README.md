# Tabmory

Minimal Chrome extension that tracks tab parent → child relationships and shows them as clickable breadcrumbs in the popup.

## Run

- Chrome → `chrome://extensions`
- Enable Developer mode
- Load unpacked → select this folder

## Use

- Click the extension icon to open the popup
- Open links in new tabs (right click → Open link in new tab) to create relationships
- Click **Refresh** to update the list
- Click a breadcrumb segment to focus that tab
